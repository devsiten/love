# Ifangbayelo Trado Medicare

Premium traditional Ifa consultancy website with bold gold branding.

**CAC REG NO: 7053384**

## Tech Stack
- Next.js 14+ App Router
- TypeScript
- Tailwind CSS
- next-intl (English & Yoruba)
- Cloudflare Pages ready

## Setup

```bash
npm install
npm run dev
```

Visit `http://localhost:3000/en` or `http://localhost:3000/yo`

## Deploy to Cloudflare

```bash
npm run deploy
```

## Features
- Bold gold business name branding
- Dark premium aesthetic
- Traditional Adinkra patterns
- Bilingual (EN/YO)
- Contact form with WhatsApp
- Mobile responsive
